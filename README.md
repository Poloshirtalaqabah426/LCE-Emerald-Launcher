No, this launcher keeps its files in a separate folder. It does not interfere with the standard Minecraft launcher or your existing worlds.

## 🔧 Troubleshooting

If the game fails to open, try these common fixes:

*   Update your graphics drivers through your computer settings.
*   Turn off your antivirus software for a few minutes to ensure it is not blocking the installation.
*   Check that you have a stable internet connection for the first-time setup.
*   Ensure you have the latest version of Java installed on your machine.

**The launcher crashes on startup:**
Check your task manager to see if another instance of the launcher is already running. Close all associated tasks and try to open the program again.

**The game loads but the screen is black:**
This usually relates to your graphics card settings. Adjust the resolution settings in the launcher menu to match your monitor.
